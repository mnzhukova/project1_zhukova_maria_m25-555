# project1_zhukova_maria_m25-555
# Лабиринт: консольная приключенческая игра

**Описание:**  
текстовая adventure игра "Лабиринт сокровищ". Игрок исследует комнаты, решает загадки, собирает предметы и пытается выбраться из лабиринта.  
Реализовано:  
- система комнат и переходов;  
- инвентарь;  
- загадки с вариативными ответами;  
- ловушки;
- случайные события

## Установка

1. Предварительные требования
На компьютере должны быть установлены:

Python 3.12+ (проверьте: python3 --version);

Poetry 1.8.4+ (проверьте: poetry --version);

git (проверьте: git --version);

make (проверьте: make --version).

Если чего‑то нет — установите через менеджер пакетов вашей ОС или официальные сайты.

2. Клонируйте репозиторий:

git clone https://github.com/<ваш-пользователь>/project1_zhukova_maria_m25-555.git
cd project1_zhukova_maria_m25-555

3. Настройка окружения

    1. Активируйте виртуальное окружение через Poetry:

    poetry env use python3
    poetry install
    Это создаст .venv и установит все зависимости из pyproject.toml.

    2. Активируйте окружение (если не активировалось автоматически):

    source $(poetry env info --path)/bin/activate

4. Сборка пакета (опционально, для публикации)

make build
Выполняет poetry build → создаёт дистрибутивы в папке dist/.

5. Установка пакета в систему
make package-install
Выполняет python3 -m pip install dist/*.whl → устанавливает пакет локально.
После этого команда project станет доступна в терминале.

6. Запуск игры

Через Poetry
poetry run project

Через Make
make project

## Как играть

go <направление> — двигаться (north/south/east/west);

take <предмет> — взять предмет;

use <предмет> — использовать предмет;

inventory — посмотреть инвентарь;

look — осмотреть комнату;

solve - попытаться решить загадку в комнате;

help — список команд;

quit/exit — выйти.
